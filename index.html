Geomagic Control X

Geomagic Control X は、簡単な操作で業界最高水準の性能を提供する計測ソフトウェア プラットフォームです。製品開発や品質管理で必要な情報をスキャンデータとCADデータの比較により、早く正確に計測結果を伝えます。また、高度なレポート作成により、計測結果を分析して品質保証や製品設計に活かせます。Geomagic Control X はあらゆる 3D デジタイザ、カメラ、スキャナに対応しています。

Geomagic Control X にするべき４つの決めて

1.検査簡素化

ユーザの検査ワークフローに合わせたプロセスの自動化から、その場で欲しい寸法を素早く計測する手軽な検査まで、製造分野の3D検査プロセスに対して対応します。また、製造部品やアセンブリの品質の向上を願うお客様の要望にお応えします。

2.検査プロセス管理

検査手順をデジタル化することで、再現性のある編集可能な検査プロセスを管理することができす。CADを活用した寸法測定が可能で、GD&T情報を含むCADフアイルの読み込みは、追加費用なく標準で提供されます。

3.検査結果分析

複数の結果を同時に比較しながら、部品間、もしくは時系列で起こる品質問題を調査することができます。さまざまな座標基準であわせた結果を同時に見れば、溶接やアセンブリで起こりうる積み上げエラーの原因を特定し、的確な対策をとるための情報を分析結果から効果的に導き出します。ポータブル測定機、光学スキャナーを使用し、関連する部品群またはアセンブリを 1 つの検査プロジェクトとして管理することで、簡単な部品間の状態も分析でき、アセンブリにおける潜在的な問題を把握できます。

4.検査レポート

グラフィカルで伝えやすいレポートを作成します。好きな視点の画像を取り込んだり、デザインを確認しながらレポートを作成できるので、従来の紙や PDF でレポートを作成した作業にとても近いものがあります。まったく新しい、自社の規定に合わせてカスタマイズした検査レポートを作成できます。3DPDFにも対応しており、3Dデータを見ながら検査結果を見ることができます。
